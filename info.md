# Crypto Card

Candlestick chart voor BTC en ETH in Home Assistant, via de Binance API.

![preview](https://img.shields.io/badge/type-Lovelace%20Card-blue)

## Features
- BTC / ETH tabs
- Huidige prijs + % change
- Candlestick chart (groen/rood)
- Intervals: 1h | 4h | 1d
- Auto-refresh elke 60 seconden
- Dark theme

## Gebruik
```yaml
type: custom:crypto-card
```
